##更新数据

主要的逻辑:

  1. 准备要更新的数据源
  2. 检查是否有LineDataSet 存在
  3. 有，则通过LineDataSet 的setValues更换整个坐标，或者 data.addEntry(…) 添加一个或者 data.removeEntry(…)删除一个。
  4. 无，则创建LineDataSet ,重新构造数据源
  5. 调用mLineChart.invalidate();更新图表
